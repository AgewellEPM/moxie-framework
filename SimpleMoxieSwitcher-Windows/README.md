## Quick Start

1. Install .NET 8.0 SDK
2. Install Docker Desktop
3. Clone repository
4. Run: `dotnet restore`
5. Run: `dotnet build`
6. Run: `dotnet run --project SimpleMoxieSwitcher/SimpleMoxieSwitcher.csproj`

## Build

```powershell
.\build.ps1 -Configuration Release -Platform x64
```
