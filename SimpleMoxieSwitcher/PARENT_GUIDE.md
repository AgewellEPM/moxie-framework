## What is SimpleMoxieSwitcher?

SimpleMoxieSwitcher is a macOS application that acts as a control center for your child's Moxie robot. Think of it like a remote control and activity selector that lets you:

- **Customize Moxie's personality** (Teacher, Coach, Storyteller, etc.)
- **Choose different activities** (Stories, Learning, Music, Languages)
- **Control Moxie's appearance** (Face expressions, LED colors)
- **View conversation history** between your child and Moxie
- **Manage settings** safely and easily

All content is stored locally on your Mac in your Library folder for privacy and safety.

## Main Features

### üë∂ Child Profile
**What it does:** Stores information about your child to personalize Moxie's interactions
**Perfect for:** Creating a tailored experience that grows with your child

**How to use it:**
1. Click "Child Profile" from the main menu
2. Enter your child's name
3. Set their birthday (optional but recommended)
4. Add interests and hobbies (e.g., "dinosaurs," "soccer," "drawing")
5. List personal goals (for your child or yourself as a parent)
6. Write notes about things you want Moxie to remember

**Why it matters:**
- Moxie uses this information to personalize every conversation
- Stories, lessons, and activities adapt to your child's interests and age
- Goals help Moxie encourage progress in specific areas
- Personal notes ensure Moxie remembers important context (allergies, preferences, etc.)

**What information is stored:**
- Name - How Moxie addresses your child
- Birthday - Used to calculate age and tailor content appropriately
- Interests - Topics Moxie will reference and build on
- Personal Goals - Areas where Moxie can provide extra support
- Things to Remember - Important context like learning challenges, siblings, etc.

**Privacy:**
All profile information is stored locally on your Mac at:
`~/Library/Application Support/SimpleMoxieSwitcher/child_profile.json`

## Detailed Feature Guide

### Creating Custom Stories

**Step-by-Step:**

1. **Start a Story**
   - Open Story Time
   - Type a topic (keep it simple: "pirates," "princess," "dinosaurs")
   - Or use the Story Wizard for guided setup

2. **Customize the Experience**
   - Click the settings icon (slider)
   - Choose reading level appropriate for your child
   - Select a genre your child enjoys
   - Pick an author style (this changes how the story is written)
   - Select fiction type

3. **During the Story**
   - Moxie creates an opening and presents 3 choices
   - Your child can pick one of the choices
   - OR type their own idea for what happens next!
   - The story grows based on their decisions

4. **Saving Stories**
   - Click "Save Story" at the bottom
   - Creates a tile you can tap to resume later
   - All saved stories appear in the Library

### Setting Up Learning Sessions

**Step-by-Step:**

1. **Choose a Topic**
   - Open Learning Lab
   - Type exactly what to learn: "multiplication tables," "planets," "Spanish verbs"
   - More specific = better results

2. **Customize Settings**
   - Click settings icon
   - Set grade level to match your child
   - Choose difficulty level
   - Select learning style (Visual learners? Choose "Visual with examples")
   - Set lesson length

3. **During Learning**
   - Moxie teaches one lesson at a time
   - Each lesson has explanations and examples
   - Quiz questions check understanding
   - Type "next" for the next lesson
   - Ask questions anytime!

4. **Saving Progress**
   - Click "Save Session"
   - Creates a tile to continue later
   - All learning sessions saved in Library

### Language Practice Sessions

**Step-by-Step:**

1. **Select Languages**
   - Open Language Lab
   - Choose Moxie's language (what Moxie speaks)
   - Choose your child's language (what your child speaks)

2. **Choose a Mode**
   - **Casual Chat** - Natural conversation practice
   - **Language Learning** - Focused learning with translations
   - **Translation Help** - Help with specific phrases

3. **Start Practicing**
   - Your child types or speaks in their chosen language
   - Moxie responds in its chosen language
   - In Learning mode, see translations for help

4. **Save Language Pairs**
   - Click "Save Language Pair" in settings
   - Quickly switch between saved combinations

### Finding and Playing Music

**Step-by-Step:**

1. **Search for a Song**
   - Type song name, artist, or even a lyric you remember
   - Moxie finds the full song lyrics

2. **Karaoke Mode**
   - Lyrics display line by line
   - Current line is highlighted and enlarged
   - Progress shows which line you're on

3. **Control Playback**
   - Use play/pause to control timing
   - Skip forward/back with arrow buttons
   - Adjust speed if too fast/slow
   - Heart icon saves to favorites

- **Conversations:** `~/Library/Application Support/SimpleMoxieSwitcher/conversations/`

- **Language Practice:** `~/Library/Application Support/SimpleMoxieSwitcher/languages/`

## Troubleshooting

### App Won't Launch
**Problem:** Double-clicking the app does nothing
**Solution:**
1. Right-click the app ‚Üí Open
2. Click "Open" in the security dialog
3. Mac may need to verify the app first time

### Can't Connect to Moxie Robot
**Problem:** Features aren't working with the physical robot
**Solution:**
1. Ensure Moxie robot is powered on
2. Check that Mac and Moxie are on the same Wi-Fi network
3. Verify MQTT broker settings in Settings
4. Default host: `192.168.1.128`, port: `1883`

### Stories/Learning Not Generating
**Problem:** "API key not set" error
**Solution:**
1. You need an OpenAI API key for these features
2. Set environment variable: `export OPENAI_API_KEY="your-key"`
3. Restart the app

### Conversations Not Appearing
**Problem:** Past conversations don't show up
**Solution:**
1. Check that conversations exist in:
   `~/Library/Application Support/SimpleMoxieSwitcher/conversations/`
2. Click the refresh button (circular arrow)
3. Make sure conversations are `.jsonl` files

### App Runs Slow
**Problem:** App is sluggish or unresponsive
**Solution:**
1. Close and reopen the app
2. Check Mac storage (app needs space for data)
3. Clear old conversation files if needed
4. Restart your Mac

### Music Lyrics Incorrect
**Problem:** Wrong song or lyrics don't match
**Solution:**
1. Make search more specific (include artist name)
2. Try searching with exact song title
3. Different versions of songs may have different lyrics

## Frequently Asked Questions

**Q: What age is this app appropriate for?**
A: The app is designed for children ages 5-13, with content ranging from preschool to adult levels. You can adjust difficulty, reading level, and complexity for any age.

**Q: Do I need an internet connection?**
A: Yes, for most features that generate content (Stories, Learning, Language, Music). The app connects to OpenAI's API and your Moxie robot over the network.

**Q: Can I use this without a Moxie robot?**
A: Yes! Many features work standalone (Stories, Learning, Language practice, Music). Robot control features require a physical Moxie robot.

**Q: How much does the OpenAI API cost?**
A: OpenAI charges per usage. Most features use the economical "gpt-4o-mini" model. Typical usage:
- A story: ~$0.01-0.05
- A learning session: ~$0.02-0.10
- Language conversation: ~$0.01-0.03 per exchange

**Q: Is my child's data safe?**
A: Yes. All data is stored locally on your Mac. OpenAI processes requests but doesn't use your data for training (per OpenAI's API data policy).

**Q: Can multiple children share one Moxie?**
A: Currently, the app supports one child profile at a time. For multiple children, you can update the child profile when switching between children, or create separate user accounts on your Mac with the app installed on each.

**Q: What happens if I don't set up a child profile?**
A: Moxie will still work perfectly! The child profile just adds extra personalization. Without it, Moxie uses general, age-appropriate responses.

**Q: Can I export conversations?**
A: Yes. All conversations are stored as `.jsonl` files in the Application Support folder. You can copy, backup, or export these files.

**Q: What languages are supported?**
A: 20+ languages including: English, Spanish, French, German, Italian, Portuguese, Russian, Chinese, Japanese, Korean, Arabic, Hindi, Dutch, Swedish, Polish, Turkish, Vietnamese, Thai, Indonesian, and Hebrew.

**Q: Can I create my own activities?**
A: Yes! Use the "Custom Creator" ‚ú® button to build custom activities (advanced feature).

**Q: How do I reset everything?**
A: Delete the folder:
`~/Library/Application Support/SimpleMoxieSwitcher/`
The app will create fresh folders on next launch.

**Q: Can I change the default settings?**
A: Yes! Click Settings ‚öôÔ∏è to configure:
- API endpoints
- MQTT broker settings
- Default personalities
- Language preferences

**Q: What if a feature isn't working?**
A: Check the Troubleshooting section above. If problems persist:
1. Restart the app
2. Check your internet connection
3. Verify API key is set
4. Check Console app for error messages

**Q: Can I suggest new features?**
A: Yes! This is a community project. Feature requests and feedback are welcome.

## Appendix: File Locations

All app data is stored in:
```
~/Library/Application Support/SimpleMoxieSwitcher/
```

Subdirectories:
- `conversations/` - All chat histories (`.jsonl` files)
- `stories/` - Saved stories (`.json` files)
- `learning/` - Learning sessions (`.json` files)
- `languages/` - Language practice sessions (`.json` files)
- `music/` - Saved favorite songs (`.json` files)
- `child_profile.json` - Your child's profile information

To backup all data:
```bash
cp -r ~/Library/Application\ Support/SimpleMoxieSwitcher/ ~/Desktop/MoxieBackup/
```

To restore from backup:
```bash
cp -r ~/Desktop/MoxieBackup/ ~/Library/Application\ Support/SimpleMoxieSwitcher/
```
